# 🍽️ Zomato Food Delivery Sales Analytics Dashboard | Power BI

## 📌 Project Overview

This project showcases an interactive **Power BI dashboard** built to analyze Zomato Food Delivery sales data and convert raw transactional records into meaningful business insights.

The dashboard follows a **Star Schema (Fact-Dimension) Data Model**, enabling efficient reporting, scalable analytics, and improved query performance. It provides stakeholders with a clear understanding of customer ordering behavior, restaurant performance, payment preferences, delivery success rates, and monthly sales trends.

Designed with interactive filters and dynamic KPIs, this dashboard helps support data-driven business decisions in the food delivery industry.

---

## 🎯 Business Objective

The objective of this project is to transform raw sales data into an interactive business intelligence solution that helps answer important questions such as:

- Which food items generate the highest demand?
- Which customer membership type contributes the most orders?
- What payment methods do customers prefer?
- How successful are deliveries compared to cancellations?
- How do sales and transactions change over time?
- Which restaurant categories perform the best?

## 🗄️ Data Model

This project follows a **Star Schema** design.

### Fact Table
**Sales Data**

Columns:
- Customer Name
- Order ID
- Order Date
- Customer ID
- Restaurant ID
- Food Item
- Quantity
- Delivery Status
- Payment Method

### Dimension Tables

- Customer Details
- Food Details
- Restaurant Details
